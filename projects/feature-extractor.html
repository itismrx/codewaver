Feature Extractor

AI that reads engineering drawings — extracting structure, dimensions, and metadata at scale.

95% extraction accuracy API-ready Supports DXF · PDF · PNG
What It Does

Automates the interpretation of engineering drawings

Using computer vision and OCR, the system identifies and extracts Bill of Materials (BOM) tables, title block fields, GD&T symbols, dimension annotations, and geometric tolerances — replacing hours of manual data entry with a single API call.

  • Zero manual data entry
  • Works on scanned and digital drawings
  • Integrates with ERP & PLM systems

BOM Extraction

Parses multi-level BOMs from drawing sheets with part numbers and quantities.

Title Block Parser

Captures part number, revision, material, scale, and drawing date fields.

GD&T Recognition

Identifies flatness, perpendicularity, true position, and circularity symbols.

Dimension Extraction

Reads linear, angular, and radial dimensions with tolerance values.

How It Works

Three steps, one API call

1

Upload Drawing

Send a PDF, PNG, or DXF file via the REST endpoint or client SDK.

2

AI Analyses Layers

Computer vision and OCR pipelines process geometry, text, and symbol layers in parallel.

3

Structured JSON Returned

All extracted data is returned as structured JSON ready for ERP or PLM ingestion.
